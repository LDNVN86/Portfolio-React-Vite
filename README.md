- 🎓 **HCMUS** — Faculty of Information Technology (Software Engineering)

- 🧰 Đang dùng hằng ngày: **PostgreSQL 16** (+ pgvector), **Redis 7**, **Centrifugo**, **Cloudflare** (Turnstile + R2), **Docker**, **GitHub Actions**, **goose**, **pgx + sqlc**

- 🗄️ Daily tools: **Linux/Ubuntu**, **DBeaver**, **Postman**, **VS Code**
